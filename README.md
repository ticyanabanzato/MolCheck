# AgroTools – MolCheck

**MolCheck** é um módulo da suíte AgroTools dedicado à análise molecular básica aplicada ao agronegócio, bioinsumos e controle de qualidade industrial.

## O que esta ferramenta faz
- Calcula GC% de primers  
- Calcula Tm básica  
- Analisa comprimento  
- Faz checagens iniciais de qualidade  
- Auxilia triagens rápidas de primers já existentes  
- Fornece alertas úteis para rotina do agro (contaminação, alvos, contextos de uso)

> Esta ferramenta não substitui o Primer3.  
> Ela complementa o workflow, permitindo avaliações rápidas e aplicadas ao contexto agroindustrial.
